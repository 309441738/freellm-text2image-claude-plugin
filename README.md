## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/freellm-text2image-claude-plugin
/plugin install freellm-text2image@freellm-text2image-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **freellm-text2image**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
